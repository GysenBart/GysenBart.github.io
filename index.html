My projects

IoT Thermostat IoT Energy Meter FreeRTOS Bootloader Matter

This project employs a 32-bit microcontroller with Ethernet for connectivity. It reads the status of a rotary encoder and transmits it to an OLED display. The software was developed with FreeRTOS and utilizes LwIP and MQTT for communication. External relays are controlled via GPIO and can be controlled via the Home Assistant app, enabling users to remotely control devices and create automation scenarios.

This project focuses on integrating IoT, electronics, and various communication technologies to create a scalable energy monitoring solution. I chose a single board computer instead of a microcontroller for more capabilities. Software programming is done remotely and cross-platform to provide an additional challenge. The energy meter is read via Modbus and the data is sent to the cloud. For visualization, I use Home Assistant, which allows me to integrate different tools like Grafana to easily visualize the data.

The smart thermostat can accurately control and monitor the temperature in a room. The OLED display gives users instant feedback on both current and desired temperatures. With WiFi and MQTT functionality, the thermostat can be controlled and monitored remotely, making it ideal for today's connected environments. This is also my first project. To mimic the operation of a heater, we used resistors that heat up when activated. The temperature sensor can then detect these temperature changes and accurately measure the ambient temperature.

This project provides a complete firmware management solution for a STM32F429ZI microcontroller. It includes a bootloader supporting both UART and UDP communication and a simple Qt software interface. The software makes firmware management easy with options for uploading and deleting firmware and a choice between UART and UDP communication. Built-in logging functionality helps with troubleshooting. The project simplifies firmware management for this type of microcontroller, suitable for development and updates. STILL WORKING ON IT.

This project is an interactive breakout game controlled by motion with a wireless controller using Matter over Thread. The breakout game is displayed on a TV using an SBC (Single Board Computer) that also serves as a Matter Hub. This was an internship project at Dekimo and the goal was to gain knowledge of the Matter protocol and to attract an audience at trade shows.
